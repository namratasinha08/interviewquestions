1. Can java be considered an absolute Object Oriented language? 
    A. True
    B. False

Answer: B

2. The conversion of Primitive data types to Object is called _______.
    A. Typecasting
    B. Boxing
    C. Serialization
    D. Synchronization

Answer: B

3. The conversion of Object to primitive datatype is called Unboxing.
    A. True
    B. False

Answer: A

4. Wrapper classes are immutable.
    A. True
    B. False

Answer: A

    B. The automatic conversion of primitive data types into its equivalent Wrapper type is known as boxing and opposite operation is known as unboxing.
