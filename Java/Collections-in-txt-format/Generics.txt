
1. It forces the Java programmer to store specific type of objects to make it type-safe which makes the code stable by detecting the bugs at ________.

    A. Compile time
    B. Run time
    C. Interrupt time
    D. Debug time

Answer: A


2. We use Generic to make it type safe.
    A. True
    B. False

Answer: A

3. Which of the following are NOT common type parameters in Generics?
    A. Value
    B. Number
    C. Element
    D. Character

Answer: D

4. Which of the following is NOT an advantage of Generics?
    A. Type-Safety
    B. Type casting is not required
    C. Type casting is required
    D. Compile -Time Checking

Answer: C

5. An invocation of a generic type is generally known as a parameterized type.
    A. True
    B. False

Answer: A

6. The _______ is known as the wildcard in generic programming.

    A. Question mark(?)
    B. Back-slash(/)
    C. Forward-slash(\)
    D. Percentage(%)

Answer: A

7. Which of the following are NOT a way of declaring wildcards in Java generics.

    A. Uper bound Wildcard
    B. Middle bound Wildcard
    C. Lower bound Wildcard
    D. Unbounded Wildcard

Answer: B

8. Best suitable wildcard conditions are based on the type of parameters passed to a method as in and out parameter.
    A. True
    B. False

Answer: A

9. Can java generics be applied to primitive types?
    A. True
    B. False

Answer: B
