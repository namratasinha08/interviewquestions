2. ______ and _______ are the classes used in Date and Time API in java 8.

   A. Local DateTime API, Remote DateTime API  
   B. Local DateTime API, Zoned DateTime API  
   C. Remote DateTime API, Zoned DateTime API  
   D. None of the above
 
Answer: B  

Explanation : Local DateTime API- Simplified form of date - time API without any complexities.
              Zoned DateTime API - Special form of date - time API  with varaiations.
